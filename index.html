About

The first dev environment faster and more secure than localhost

LocalWebContainers are powered by Service Workers which boots entire development environments instantly and securely within your browser. You don't have to start a web server to get a fully functional web page up and running.

Every subdomain is it's own sandboxed origin set up for your own project. Nobody with the same link will see what you can see. Everything served on the domains is loaded entirely from your browser

Plugins

More than just a simple sandbox!

With localwebcontainer your are not only getting a static web page, with our built in plugins we render markdown and compile typescript JIT so you can be sure that your project will be up and running in no time without any build tools or any other dependencies.
